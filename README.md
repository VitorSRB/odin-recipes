# Odin Recipes 🍳

This is my first project from **The Odin Project's** Full Stack curriculum. The main goal of this application is to demonstrate mastery of fundamental HTML concepts before moving on to styling and advanced layouts.

### 🚀 Skills Demonstrated
* Basic HTML structure (`doctype`, `html`, `head`, `body`).
* Working with headings, paragraphs, and lists (ordered and unordered).
* Managing internal and external links.
* Embedding images and handling file paths.
* Semantic HTML usage.

### 🛠️ Technologies Used
* **HTML5** (Core structure)
* **Git/GitHub** (Version control)
